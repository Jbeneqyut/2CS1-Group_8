# PART 9: Reflection Questions

### 1. What is the difference between commit and push?
A commit saves your changes to your local repository, while a push uploads those committed changes to a remote repository such as GitHub.

   ---
   
### 2. Why are branches important?
Branches are important because they allow developers to work on different features or fixes at the same time without affecting the main codebase.

---
   
### 3. What causes merge conflicts? 
Merge conflicts occur when two branches modify the same part of a file and Git cannot automatically decide which changes to keep.
